VIDEO RECORDING GUIDE (1-2 Minutes)
====================================

This guide helps you create a professional demo video for your YouTube submission.

VIDEO REQUIREMENTS
------------------
Duration: 1-2 minutes
Platform: YouTube (public or unlisted)
Quality: 720p or higher recommended
Audio: Optional (can be silent or with explanation)

WHAT TO SHOW
------------

PART 1: DESKTOP VIEW (30-40 seconds)
-------------------------------------
1. Show the landing page
   - Full table view
   - Clean, professional design
   - All token columns visible

2. Demonstrate sorting
   - Click on "Price" header
   - Click again to reverse
   - Show arrow indicators changing
   - Click "Market Cap" header
   - Show data reordering

3. Demonstrate filtering
   - Click "Filters" button
   - Select "New Pairs"
   - Show filtered results
   - Select "Verified Only"
   - Show combined filters
   - Click "Clear All"

4. Demonstrate search
   - Type in search box
   - Show instant filtering
   - Clear search

5. Show interactive elements
   - Hover over verified badge (tooltip appears)
   - Click market cap value (popover opens)
   - Click holder count (modal opens)
   - Close modal
   - Copy token address (show copied message)

PART 2: REAL-TIME UPDATES (15-20 seconds)
------------------------------------------
1. Pause for a moment
2. Point out or circle a few prices
3. Let viewer see prices update
4. Highlight the flash animations:
   - Green flash for price increase
   - Red flash for price decrease
5. Show smooth color transitions

PART 3: RESPONSIVE DESIGN (30-40 seconds)
------------------------------------------
1. Start resizing browser window
   - Slowly drag from right edge
   - Show how layout adapts
   - Pause at tablet size (768px)
   - Continue to mobile size

2. Show mobile view (< 768px)
   - Card-based layout
   - All information still visible
   - Scroll through cards
   - Show search still works
   - Show filters still work

3. Optionally resize back to desktop
   - Show smooth transition
   - Everything still works

OPTIONAL ENHANCEMENTS
---------------------
- Add background music (soft, non-intrusive)
- Add voice-over explanation
- Add text overlays highlighting features
- Add cursor highlights (yellow circle around cursor)
- Add zoom-in on key interactions

RECORDING TIPS
--------------
1. Close unnecessary applications
2. Clear browser cache/restart browser
3. Use incognito/private mode (clean, no extensions)
4. Maximize window or use full screen
5. Hide bookmarks bar (Ctrl/Cmd + Shift + B)
6. Set browser zoom to 100%
7. Practice run before recording
8. Speak clearly if adding voice-over
9. Keep cursor movements smooth
10. Don't rush - show features clearly

- Jump cut between sections
