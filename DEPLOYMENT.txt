TOKEN TRADING TABLE - DEPLOYMENT GUIDE
======================================

OVERVIEW
--------
A pixel-perfect replica of Axiom Trade's token discovery table with real-time updates,
built with Next.js 14, TypeScript, Tailwind CSS, Redux Toolkit, and React Query.

FEATURES
--------
✓ All token columns (New pairs, Final Stretch, Migrated)
✓ Interactive elements: Popover, Tooltip, Modal, Sorting
✓ Real-time price updates with WebSocket mock
✓ Loading states: Skeleton, Shimmer, Progressive loading, Error boundaries
✓ Fully responsive (320px to desktop)
✓ Atomic architecture for maximum reusability
✓ Performance optimized with memoization and infinite scroll

TECH STACK
----------
- Next.js 14 (App Router)
- TypeScript (Strict mode)
- Tailwind CSS
- Redux Toolkit (State management)
- React Query (Data fetching)
- Radix UI (Accessible components)
- Lucide React (Icons)

INSTALLATION
------------
1. Clone the repository:
   git clone <your-repo-url>
   cd ASSIGNMENT

2. Install dependencies:
   npm install
   (or yarn install / pnpm install)

3. Run development server:
   npm run dev

4. Open http://localhost:3000 in your browser

DEPLOYMENT TO VERCEL
--------------------
1. Push your code to GitHub:
   git init
   git add .
   git commit -m "Initial commit: Token Trading Table"
   git remote add origin <your-github-repo-url>
   git push -u origin main

2. Deploy to Vercel:
   - Visit https://vercel.com
   - Click "Import Project"
   - Select your GitHub repository
   - Vercel will auto-detect Next.js and configure build settings
   - Click "Deploy"
   - Your app will be live at: https://your-project.vercel.app

ALTERNATIVE DEPLOYMENT (Manual)
-------------------------------
1. Install Vercel CLI:
   npm install -g vercel

2. Deploy:
   vercel
   (Follow the prompts)

3. For production:
   vercel --prod

KEY FEATURES IMPLEMENTED
------------------------
1. INTERACTIVE COMPONENTS
   - Tooltip: Hover over verified badges and info icons
   - Popover: Click market cap for detailed breakdown
   - Modal: Click holders count for detailed stats
   - Sorting: Click any column header to sort
   - Filters: Status, verified, trending filters

2. REAL-TIME UPDATES
   - WebSocket mock simulates live price updates
   - Smooth color transitions on price changes
   - Flash animations (green for up, red for down)

3. LOADING STATES
   - Skeleton screens during initial load
   - Shimmer effect for better UX
   - Infinite scroll for progressive loading
   - Error boundaries for graceful error handling

4. PERFORMANCE OPTIMIZATIONS
   - Memoized components with React.memo
   - Virtualized rendering with intersection observer
   - Debounced search input
   - Optimized re-renders with Redux selectors
   - Next.js image optimization ready
   - Code splitting with dynamic imports

5. RESPONSIVE DESIGN
   - Desktop: Full table view with all columns
   - Tablet: Adjusted spacing and font sizes
   - Mobile: Card-based layout (320px+)
   - Touch-friendly interactions
   - Optimized for all screen sizes

PERFORMANCE METRICS
-------------------
Target: Lighthouse score ≥ 90 on mobile & desktop

Optimization techniques:
- React.memo for expensive components
- useCallback for event handlers
- Efficient Redux selectors
- Intersection Observer for infinite scroll
- CSS transitions over JavaScript animations
- Tailwind CSS for minimal CSS bundle
