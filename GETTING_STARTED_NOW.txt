COMMON ISSUES & SOLUTIONS
--------------------------

Issue: "npm install" fails
Solution: Delete node_modules and package-lock.json, try again

Issue: Port 3000 already in use
Solution: Run on different port: npm run dev -- -p 3001

Issue: Build fails
Solution: Run "npm run lint" to check for errors

Issue: Styles not loading
Solution: Delete .next folder: rm -rf .next (or manually)
